How to run program:
1. gcc --std=gnu99 -pthread -lm -o line_processor Multithreaded-Pipeline.c
2. ./line_processor <input1.txt> output.txt      or      ./line_processor <input1.txt
    a. look at output to see results
  	b. output will show the input entered in
  	c. output will show sentence with \n replaced with ' '
  	d. output will show ++ replaced with ^
  	e. output will show lines with 80 characters each

When inputing input.txt by hand
3. ./line_processor
4. Then It will asked for your input
5. Paste in input
6. Press Enter
7. Then press Ctrl+D
8. Follow step 2(a,b,c,d,e)
